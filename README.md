# To-Do List Application

A simple command-line based To-Do List Application built using Python. This project allows users to add tasks, view saved tasks, and exit the application through an interactive menu system.

 # Features

* Add new tasks
* View all added tasks
* Simple and beginner-friendly interface
* Runs continuously until the user chooses to exit

##  Technologies Used

* Python

##  How It Works

1. The program displays a menu with options.
2. Users can:

   * Add a task
   * View existing tasks
   * Exit the program
3. Tasks are stored temporarily in a Python list during program execution.

##  How to Run

1. Install Python on your system.
2. Save the code in a file named `todo.py`
3. Open terminal or command prompt.
4. Run the program using:

```bash
python todo.py
```
